---------------------------------------------------------------------------------
Список файлов:

    * spschedule - библиотека для работы с расписанием
    | - sat_schedule.h(cpp) - составление расписания, решение конфликтов.
    | - session.h(cpp) - сеанс;
    | - scheduleinfo.h(cpp) - информация для составления расписания;
    | - schedule.h(cpp) - расписание;
    | - schedulefile.h(cpp) - файл распсиания;
    | - scheduler.h(cpp) - планировщик расписания.
